| 🚢 船 | 300 | >10级 | 解锁海洋钓鱼 |

## 三、象棋游戏优化

### 3.1 每日对局奖励系统
- 每日可领取5次奖励，需要完成相应对局次数
- 第1局完成 → 可领第1次（20积分）
- 第2局完成 → 可领第2次（30积分）
- 第3局完成 → 可领第3次（40积分）
- 第4局完成 → 可领第4次（50积分）
- 第5局完成 → 可领第5次（80积分）
- 总计：220积分/天

### 3.2 样式优化
- 改为亮色系，与数独游戏风格一致
- 背景使用渐变色彩
- 按钮和卡片样式统一

### 3.3 走棋动态效果
- 使用 requestAnimationFrame 实现平滑移动动画
- 棋子移动有过渡效果

### 3.4 关键修改文件
- **chess-game.html**：添加每日对局计数和奖励逻辑
- **chess.html**：添加奖励面板UI
- **chess-game.css**：更新样式为亮色系

## 五、技术要点

### 5.1 localStorage 数据结构
- `fishing_daily`：存储每日钓鱼数据 `{ date, pondCount, oceanCount }`
- `chess_daily`：存储象棋每日对局数据 `{ date, gamesPlayed, rewardsClaimed }`
- 玩家数据中的 `items.universal_fragment`：万能碎片数量

### 5.2 日期重置机制
- 使用 `new Date().toDateString()` 判断是否新的一天
- 日期不同时自动重置数据

### 5.3 动态下拉实现
- 使用 CSS `max-height` 和 `transition` 实现平滑动画
- JavaScript 控制 `expanded` 类的添加/移除

## 六、注意事项

1. **等级判断**：钓竿和船的等级限制使用 `playerLevel > LEVEL_LIMITS[itemId]`，即超过指定等级才显示
2. **已拥有道具**：即使等级不足，已拥有的道具仍会显示"已拥有"状态
3. **碎片不浪费**：万能碎片可以累积，不会因为已拥有道具而浪费
4. **弹窗刷新**：合成后调用 `showShopModal()` 刷新商城数据
5. **首页刷新**：关闭商城弹窗时调用 `location.reload()` 刷新首页
